## Installation

```bash tab="pnpm"
pnpm add @deltakit/react
```

```bash tab="npm"
npm install @deltakit/react
```

```bash tab="yarn"
yarn add @deltakit/react
```

This installs both `@deltakit/react` and `@deltakit/core` (its dependency). You only need to import from `@deltakit/react`.

## Requirements

- React 18 or higher
- A backend endpoint that streams Server-Sent Events (SSE)

## Minimal Example

```tsx
import { useStreamChat } from "@deltakit/react";

function Chat() {
  const { messages, isLoading, sendMessage } = useStreamChat({
    api: "/api/chat",
  });

  return (
    <div>
      {messages.map((msg) => (
        <div key={msg.id} style={{ margin: "8px 0" }}>
          <strong>{msg.role}:</strong>{" "}
          {msg.parts
            .filter((p) => p.type === "text")
            .map((p) => p.text)
            .join("")}
        </div>
      ))}

      {isLoading && <span>Thinking...</span>}

      <button
        onClick={() => sendMessage("Hello!")}
        disabled={isLoading}
      >
        Send
      </button>
    </div>
  );
}
```

## Server Requirements

Your backend needs to:

1. Accept `POST` requests with a JSON body containing `{ message: string }`.
2. Respond with `Content-Type: text/event-stream`.
3. Stream SSE events in `data: {JSON}\n\n` format.
4. Send `data: [DONE]\n\n` when the stream is complete.

### Expected Request

```json
{
  "message": "User's message text"
}
```

Any additional fields from the `body` option are merged in:

```json
{
  "message": "User's message text",
  "model": "gpt-4",
  "sessionId": "abc123"
}
```

### Expected Response

```
data: {"type":"text_delta","delta":"Hello"}

data: {"type":"text_delta","delta":" there!"}

data: [DONE]
```

### FastAPI Example

```python
from fastapi import FastAPI
from fastapi.responses import StreamingResponse
import json

app = FastAPI()

@app.post("/api/chat")
async def chat(request: dict):
    message = request["message"]

    async def generate():
        # Replace with your LLM call
        for word in ["Hello", " there", "!"]:
            yield f'data: {json.dumps({"type": "text_delta", "delta": word})}\n\n'
        yield "data: [DONE]\n\n"

    return StreamingResponse(generate(), media_type="text/event-stream")
```
