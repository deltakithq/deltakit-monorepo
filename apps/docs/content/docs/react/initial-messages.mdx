---
title: Initial Messages
description: Pre-populate conversations from a database or API.
---

Use the `initialMessages` option to load an existing conversation when the component mounts.

## Basic Usage

```tsx
import { useStreamChat } from "@deltakit/react";

function Chat() {
  const { messages, sendMessage } = useStreamChat({
    api: "/api/chat",
    initialMessages: [
      { id: "1", role: "user", parts: [{ type: "text", text: "Hello" }] },
      { id: "2", role: "assistant", parts: [{ type: "text", text: "Hi there! How can I help?" }] },
    ],
  });

  // messages starts with the two messages above
}
```

## Loading from an API

Fetch conversation history from your backend and pass it as `initialMessages`:

```tsx
import { useStreamChat } from "@deltakit/react";
import { useState, useEffect } from "react";

function Chat({ sessionId }: { sessionId: string }) {
  const [history, setHistory] = useState([]);

  useEffect(() => {
    fetch(`/api/chat/${sessionId}/history`)
      .then((res) => res.json())
      .then(setHistory);
  }, [sessionId]);

  const { messages, sendMessage } = useStreamChat({
    api: "/api/chat",
    initialMessages: history,
    body: { sessionId },
  });

  return (/* render messages */);
}
```

## Converting OpenAI Agents SDK History

If your backend uses the OpenAI Agents SDK, the history format (`TResponseInputItem[]`) won't match DeltaKit's `Message[]` directly. Use `fromOpenAiAgents` to convert:

```tsx
import { fromOpenAiAgents, useStreamChat } from "@deltakit/react";

async function fetchHistory() {
  const res = await fetch("/api/chat/");
  const openAiItems = await res.json();
  return fromOpenAiAgents(openAiItems);
}

function Chat() {
  const [initialMessages, setInitialMessages] = useState([]);

  useEffect(() => {
    fetchHistory().then(setInitialMessages);
  }, []);

  const { messages, sendMessage } = useStreamChat({
    api: "/api/chat",
    initialMessages,
  });

  return (/* render messages */);
}
```

`fromOpenAiAgents` handles all the complexity of mapping agent items — user messages, function calls, tool outputs, reasoning — into clean `Message<ContentPart>[]`. See [fromOpenAiAgents](/docs/core/from-openai-agents) for the full mapping table.

## Notes

- `initialMessages` is only read on the first render. Changing it after mount has no effect.
- To reset the conversation with new messages, use `setMessages` instead:

```tsx
const { setMessages } = useStreamChat({ api: "/api/chat" });

// Reset with new history
setMessages(newMessages);
```
